# Contributing to Lyra Reflex

We welcome contributions to the neural-audio edge!

## Workflow
1. Fork the repo.
2. Create a feature branch (`git checkout -b feature/neural-engine`).
3. Commit changes (ensure Ansible playbooks are validated).
4. Push to the branch.
5. Open a Pull Request.

## Areas for Contribution
- **DSP**: New PipeWire routing optimizations.
- **AI**: Hailo-8 model optimizations (HEF files).
- **IaC**: Ansible role improvements for newer kernels.
- **RF**: Improved RDS modulation logic.

Please follow the coding standards established in the existing `.tsx` and `.yml` files.
